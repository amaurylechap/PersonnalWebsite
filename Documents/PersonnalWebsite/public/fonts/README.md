# Neue Montreal Font Files

Place your Neue Montreal font files here.

## Required Files

Place the following font files in this directory:

- `NeueMontreal-Regular.woff2` (or `.otf`)
- `NeueMontreal-Medium.woff2` (or `.otf`)
- `NeueMontreal-Bold.woff2` (or `.otf`)

## Optional Files

- `NeueMontreal-RegularItalic.woff2` (or `.otf`)
- `NeueMontreal-MediumItalic.woff2` (or `.otf`)

## Where to Get Neue Montreal

Neue Montreal is available from **Pangram Pangram Foundry**:
- Website: https://pangrampangram.com/products/neue-montreal
- Free trial available for personal use
- Commercial license required for commercial projects

## Formats

**Preferred format**: WOFF2 (best compression, modern browsers)
**Fallback**: OTF or TTF (works everywhere)

## After Adding Fonts

Once you add the font files to this directory (`public/fonts/`), the website will automatically use Neue Montreal. If the files are not found, it will fall back to Inter font.

## File Naming

Make sure the file names match exactly:
- `NeueMontreal-Regular.woff2`
- `NeueMontreal-Medium.woff2`
- `NeueMontreal-Bold.woff2`

If your files have different names, update the paths in `app/globals.css`.
